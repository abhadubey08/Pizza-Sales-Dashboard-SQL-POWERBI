# Pizza Sales Analysis Dashboard | SQL & Power BI

## Project Overview

This project focuses on analyzing pizza sales data to uncover business insights related to revenue generation, customer purchasing behavior, product performance, and sales trends. SQL was used for data extraction and analysis, while Power BI was used to build an interactive dashboard for business reporting and decision-making.

---

## Objectives

* Analyze overall sales performance
* Identify top and bottom performing pizzas
* Understand customer ordering patterns
* Evaluate category and size-wise sales contribution
* Generate actionable business insights through data visualization

## Key Business Insights

* Weekend evenings generated the highest order volume.
* July recorded the highest monthly sales activity.
* Classic pizzas contributed the highest sales and order volume.
* Large-sized pizzas generated the highest revenue contribution.
* Low-performing products were identified for menu optimization and inventory planning.
